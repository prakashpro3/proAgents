## Available Prompts

| Prompt | Use Case |
|--------|----------|
| [analyze.md](./analyze.md) | Analyze performance issues |
| [frontend.md](./frontend.md) | Frontend optimization |
| [backend.md](./backend.md) | Backend optimization |
| [database.md](./database.md) | Database optimization |

---

## Quick Commands

```bash
pa:perf analyze <code>       # Analyze code performance
pa:perf profile <function>   # Profile function execution
pa:perf optimize <target>    # Suggest optimizations
pa:perf benchmark <code>     # Create benchmarks
```
